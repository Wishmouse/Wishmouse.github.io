It all looks Greek to me....

()

Parentheses: These guys are used to mark to contain a list of parameters, or surround conditional statements

{}

Braces: These are used to group statements, the mark the beginning and the end of a function and separate out code within the function, braces help understand the structure of the script and what we want it to do..

[]

Brackets: Are used to mark and index into an array, an array is a list

" "

Single quotes and double quotes: these are used for characters eg. ‘a’ and double quotes for a string eg “can’t beat Wellington on a good day” . However, if you are closing a few words of text it doesn’t really matter which one you use, just be consistent.
