What does it all mean?

Think of code as decorating a room
  • HTML will be the furniture, ornaments and rugs.
  • CSS with be the colour scheme, style of the fittings and their size (eg. single chairs or a large couch).
  • DOM will be how the room is put together, how the items arranged in the room.
HTML is the building-blocks of web pages. HTML allows you to put images, text, videos, forms and other pieces of content together into a cohesive webpage. You can also tweak words, resize images, and add links. Anything that you can read or view on this website is being presented to you (by way of a browser) with HTML. CSS dictates your websiteâ€™s look and feel. Font size, font colour, font type, styling around images, page layout, mouse-over effects and more are all determined by CSS. DOM provides a structured layout of the document (a tree) it defines how the text and images will be laid out on the page, it give the webpage structure.
So what is BOXIFY Everything on your webpage is arranged in boxes and boxifying you design lets you organise your page content into separate dedicated boxes for each specific section. When creating you page, you lay all the elements out and draw boxes around them to show how they will fit on a page. The size of each box is in your complete control but it recommended to divide the page into rows with 12 columns, which makes 12 boxes, this give you the most flexibility in your layout.
The Box Model The CSS box model is essentially a box that is around all HTML elements. It consists of: margins, borders, padding, and then the actual. There are four elements to the box model:
  • Content - The content of the box, where text and images appear
  • Padding - Clears an area around the content, it can be made larger or smaller leaving a gap between text and the border. The padding is transparent
  • Border - A border that goes around the padding and content. Borders can be given a colour / style
  • Margin - Clears an area outside the border. The margin is transparent - Clears an area outside the border. The margin is transparent
