Array: .map() .filter() .reduce()

.map() are used in array’s. It is used to map to reference an existing array then create a new array containing changes. The first array remains unchanged.

.filter() again, used in array’s, they produce an array with elements from the created by a function in the form of an array. It is a group of processes based on an array. The original array remains unchanged.

.reduce() is used to reduce an array to one value.
