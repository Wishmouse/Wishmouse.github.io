Global vs local scope

Scope! refers to the part of the code that has access to a variable. Variables defined outside a function are […] called global variables and are accessible through the code below the point of insertion, pretty much like Mastrcard, its accepted everywhere. Little example below:

var global = 10;

Variables defined within a function and inside curly brackets are local variables and can only be used inside the function, var local is not accessible anywhere else. The curly brackets make it invisible (oooh the mystery). And look, another example,

function fun() {

var local = 5;

}
